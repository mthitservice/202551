## Voraussetzungen

### Für Teilnehmer
- Windows 10/11 oder Windows Server 2019/2022
- PowerShell 5.1 oder PowerShell 7.x
- Visual Studio Code mit PowerShell Extension
- Administrator-Rechte auf dem Lab-System
- Grundkenntnisse in PowerShell (Cmdlets, Variablen, Schleifen)

### Für Trainer
- Alle Teilnehmer-Voraussetzungen
- Zweiter Monitor empfohlen
- Zugriff auf Active Directory (für JEA-Demos)

## Hinweise für den Trainer

1. **Vor jeder Demo:** Demo-Script durchgehen und ggf. an lokale Umgebung anpassen
2. **Während der Übungen:** Aktiv durch den Raum gehen und bei Fragen helfen
3. **Nach den Übungen:** Gemeinsame Besprechung der Lösungen
4. **Pausen einhalten:** Regelmäßige Pausen sind wichtig für die Konzentration
